# Grid Quest

Grid Quest is a browser-based arcade learning game about electricity, artificial intelligence (AI), data centers, and how AI can influence city energy demand.

## Play locally

Open `index.html` in a browser.

## Contents

- `index.html` - the game
- `assets/` - generated game art and sprites
- `templates/` - reusable game creation templates
- preview images for the final puzzle board

## Notes

All game visuals are original generated assets created for this project.
